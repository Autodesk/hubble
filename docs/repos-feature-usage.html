Repository Feature Usage

The percentage of active repositories located in organizations, having at least one active protected branch, and having at least one topic assigned to it.

In this context, active designates repositories with at least one pushed commit in the last four weeks.

All shared branches (e.g. master or release branches) should be protected to ensure that they cannot accidentally be deleted or rewritten. Furthermore, it can be useful to describe the nature of repositories by using topics.

Repository Size

Shows how many repositories are larger than 1GB.

GitHub recommends to keep the repository size below 1 GB. Repositories above that limit require usually more extensive resources for the automated repository maintenance which can affect the overall health of the appliance.

You can reach out to GitHub Professional Services if you want help to shrink your repositories.

Users (Active/Using License)

Using license shows you how many GitHub Enterprise licenses you use on a given day. Pushing commits shows you how many different people pushed commits to your appliance in the last month/week/day. The gap to using license indicates the amount of people that have either not logged in or that have not pushed code (e.g. people that only read/comment code).
